Project Title: AI-Driven Natural Language Test Suite Architect

Description: A tool engineered to transform product requirement descriptions into structured JSON test suites (Happy Path, Edge Cases, Security Boundaries) using LLM context mapping.

Stack: Python, OpenAI SDK, Python-Dotenv.

Installation/Usage: Brief steps on running pip install -r requirements.txt and setting up the .env file (mentioning that credentials are excluded for security).

Architecture: The CrewAI multi-agent workflow (5 agents / 5 tasks) is documented in [test_suite_crew/AGENTS_DIAGRAM.md](test_suite_crew/AGENTS_DIAGRAM.md).
